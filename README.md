# Anas Allouch - AI & Cybersecurity Portfolio

This is a premium portfolio website built with Next.js, Tailwind CSS, and Framer Motion, designed with a "Google Anti-Gravity" aesthetic.

## Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Font**: Inter & Roboto (via `next/font`)

## Getting Started

### 1. Install Dependencies

First, ensure you have Node.js installed. Then, install the project dependencies:

```bash
npm install
```

### 2. Run Development Server

To start the local development server with hot-reloading:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### 3. Build for Production

To create an optimized production build:

```bash
npm run build
```

Then, you can start the production server:

```bash
npm start
```

## Project Structure

- `src/app`: Main application pages and global styles.
- `src/components/layout`: Global layout components (Navbar, Footer).
- `src/components/sections`: Individual page sections (Hero, About, Projects, etc.).
- `src/components/ui`: Reusable UI components.

## Features

- **Anti-Gravity Effects**: Floating background elements and smooth animations.
- **Glassmorphism**: Modern, translucent UI elements.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **SEO Optimized**: Metadata and semantic HTML structure.
